Calculadora de IMC - JavaScript

Calculadora de IMC com as seguintes verificações

- IMC	Classificação
- Menor que 18,5	Magreza
- 18,5 a 24,9	Normal
- 25 a 29,9	Sobrepeso 
- 30 a 34,9	Obesidade grau I
- 35 a 39,9	Obesidade grau II
- Maior que 40	Obesidade grau III
